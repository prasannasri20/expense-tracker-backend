# Expense Tracker Backend

A backend project built using **Spring Boot, Spring Security, JWT, MySQL, and JPA/Hibernate** for managing personal income and expense transactions.

## Features
- User Registration
- User Login with JWT Authentication
- Secure Transaction APIs
- Add Transaction
- Get All Transactions
- Get Transaction By ID
- Update Transaction
- Delete Transaction
- Each transaction is linked to the logged-in user

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA / Hibernate
- MySQL
- Maven
- Postman
